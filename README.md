# Hotel_reservation_system
Projekt studencki w ramach przedmiotu Metody tworzenia aplikacji bazodanowych. Ma na celu stworzenie pełnego systemu rezerwacji hotelowych (backend + frontend + baza danych), w którym klienci mogą przeglądać dostępne pokoje, dokonywać rezerwacji, płacić (symulacja prostych operacji związanych z płatnością), a administratorzy zarządzać obiektem.

Kto sklonuje repo ma użyć .env.example jako wzoru i zrobić własny .env
